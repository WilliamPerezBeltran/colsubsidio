# Vivienda Colsubsidio — Conversión y Optimización Digital

Prueba técnica para la posición de **Analista de Conversión y Optimización Digital** en Vivienda Colsubsidio: diagnóstico de datos, framework de KPIs, diseño de prueba A/B y plan de acción, construidos sobre el funnel de WhatsApp y Landing de enero 2026.

**Ver en vivo:** [williamperezbeltran.github.io/colsubsidio/informe.html](https://williamperezbeltran.github.io/colsubsidio/informe.html)

## Contenido

- **`informe.html`** — el entregable. Un wizard interactivo de 21 pasos (HTML/CSS/JS autocontenido, sin build ni dependencias externas) que recorre: el problema, diagnóstico del funnel, KPIs y esquema de control, diseño de la prueba A/B, y el plan de acción priorizado.
- **`informe.pdf`** — versión exportada de `informe.html`, para lectura sin conexión o impresión.
- **`prueba/pt_analista_conversion_digital.pdf`** — el caso original de la prueba técnica.
- **`docs/`** — notas de trabajo (transcripción del caso, borradores por paso, preguntas de entrevista). No son parte del entregable formal.

## Cómo verlo

Abrir `informe.html` directamente en un navegador (no requiere servidor ni instalación), o visitar el link publicado arriba vía GitHub Pages.

## Los números base

Todo el diagnóstico se deriva de los datos crudos del funnel de enero 2026:

- **WhatsApp:** 8.200 conversaciones → 2.700 autogestión / 5.500 desbordadas → 2.900 contactados → 607 agendados → 13 ventas.
- **Landing:** 48.000 sesiones → 3.200 iniciados → 1.850 completados → 1.200 contactados → 210 agendados → 8 ventas.
